# C-Exercise4.9
C程序设计 （第四版） 谭浩强 习题4.9

## 题目描述

给一个不多于5位的正整数，要求：
1. 求出它是几位数；
2. 分别输出每一位数字；
3. 按逆序输出各位数字，例如原数为321，应输出123。
